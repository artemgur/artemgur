Выполнены задания 2 и 3.
Задания выполнено на C#.
Программа определяет покупательскую способность товаров, а также находит сопутствующие товары для товара, заданного в файле settings.txt.
По вычисленным результатам программа строит графики.
Если файла нет, программа производит вычисления для товара, находящегося в середине отсортированного по покупательской способности списка.
Вычисления производятся на основе файлов csv. Программа работает с первым файлом этого типа, находящимся в папке с exe.
Если в папке больше 1 csv файла, игнорируются все csv файлы, кроме одного.
В DataMiningTest\bin\Debug есть скомпилированный exe файл.
